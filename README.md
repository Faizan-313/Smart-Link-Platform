# Smart Link Platform

A lightweight link management project that starts as a URL shortener and will grow into a full suite of link-related tools.

## Core Features

- URL shortener

## Project Structure

- `client/` — Frontend application built with Vite and React/TypeScript
- `server/` — Backend API and server logic implemented in TypeScript

## Getting Started

1. Install dependencies for both `client` and `server`.
2. Create env files for both client and server.
2. Run the backend server from `server/`.
3. Run the frontend client from `client/`.

## Future Enhancements

This project will expand beyond URL shortening to a broader link platform, including:

- QR code generation and download
- Link categorization and tagging
- Secure access control for link sharing
- Short link reporting and statistics
- URL redirect management

## Notes

This repository is an evolving project where link-related features are being added iteratively.
